## How to use in Cursor

1. Unzip and open this folder in Cursor (File → Open Folder)
2. Turn ON Codebase Indexing in Cursor Settings → Features
3. Open Composer (Ctrl+I / Cmd+I)
4. Say: "Read index.html and styles.css completely. Wait for my instructions."
5. Run prompts in this order:

   STEP 1: Paste CURSOR_BUILD_ALL.md (builds the 90 main pages)
   STEP 2: Paste PROMPT_1_CUSTOM_TRUCK.md (adds Custom Truck One Source, Altec, Elliott)
   STEP 3: Paste PROMPT_2_DATA_CENTER.md (adds full data center infrastructure page)
   STEP 4: Paste PROMPT_3_MRI_ROOM.md (adds RF shielding room costs, imaging center startup guide)
   STEP 5: Paste PROMPT_4_STARTUP_GUIDES.md (adds 5 business startup guide pages)
   STEP 6: Paste PROMPT_5_MISSING_MANUFACTURERS.md (fills all manufacturer gaps)

6. If Cursor stops mid-build, type: "Continue building from where you left off"
