## How to use in Cursor

1. Unzip and open this folder in Cursor (File → Open Folder)
2. Make sure Codebase Indexing is ON in Cursor Settings → Features
3. Open Composer (Ctrl+I / Cmd+I)
4. Type: "Read index.html and styles.css completely. Do not confirm, just read them and wait."
5. Once confirmed, paste the full contents of CURSOR_BUILD_ALL.md
6. If Cursor stops mid-build, type: "Continue building from where you left off"
7. Repeat until all 90 pages are built
